# 🎨 設計系統整合指南

## 什麼是 awesome-design-md？

**awesome-design-md** 是一個精選的設計系統集合，包含 55+ 個知名網站的 DESIGN.md 文件。

### 為什麼有用？

- 📋 **即用型設計系統**：複製 DESIGN.md 到項目中
- 🤖 **AI 友好**：設計代理可以讀取並生成一致的 UI
- 🎨 **像素級準確**：基於真實網站的設計規範
- 📚 **55+ 個品牌**：包括 Claude、Apple、Airbnb 等

## 🚀 如何使用

### 1. 瀏覽可用的設計系統

已克隆到：`C:\Users\user\Projects\awesome-design-md`

包含的設計系統：

### 3. 應用到項目中

**選項 A：直接複製**
```bash

# 複製預覽文件

```

**選項 B：自訂整合**
```bash

```

## 📚 進階資源

### 設計系統結構

每個 DESIGN.md 包含：
1. **視覺主題**：氛圍、設計哲學
2. **顏色系統**：語義名稱 + 功能角色
3. **排版規則**：字體層次表
4. **組件樣式**：按鈕、卡片、輸入框等
5. **佈局原則**：間距、網格、留白
6. **深度與層次**：陰影系統
7. **設計準則**：Do's and Don'ts
8. **響應式行為**：斷點、觸控目標
9. **代理提示指南**：快速參考

### 預覽文件

每個設計系統都包含：
- `preview.html` - 視覺目錄（淺色模式）
- `preview-dark.html` - 視覺目錄（深色模式）
