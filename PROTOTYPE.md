# NGO GPS Orienteering App - Prototype

## 🎯 專案概述

這是一個為 NGO 定制的定向運動應用程式，讓訪客可以通過探索地點、掃描 QR 碼來積分，並兌換紀念品。

## 🚀 線上版本

**生產環境：**
```
https://cozy-gnome-562896.netlify.app
```

## ✨ 核心功能

### 主辦方功能
- ✅ 預生成隨機登錄碼或管理用戶帳戶
- ✅ 預設和管理活動（啟用/停用、添加簽碼點）
- ✅ 預設和管理兌換商店（產品照片/名稱/所需分數/描述/數量）
- ✅ 查看所有用戶分數，手動調整
- ✅ 手動記錄分數兌換
- ✅ 管理公告板，向用戶廣播消息

### 用戶功能
- ✅ 通過手機短信或主辦方生成的代碼註冊
- ✅ 選擇活動並進入地圖查看簽碼點
- ✅ 前往地點並掃描 QR 碼
- ✅ 查看當前分數
- ✅ 查看公告板
- ✅ 查看兌換頁面（詳情僅展示，兌換由主辦方在櫃檯手動完成）

### 可選功能
- ✅ GPS 感應，附近時自動彈出簽碼點
- ✅ 拍照並審查/提交照片作為簽碼點任務
- ✅ 多項選擇題，提交後顯示正確答案
- ✅ 主辦方更新的兌換記錄

## 🛠️ 技術堆疊

- **前端**: React 18 + TypeScript
- **構建工具**: Vite
- **地圖**: Leaflet + React Leaflet
- **QR 掃描**: html5-qrcode
- **路由**: React Router v7
- **存儲**: LocalStorage（數據持久化）
- **PWA**: 支持離線使用和安裝到手機桌面

## 📱 如何使用

### 主辦方登入
1. 訪問網站
2. 點擊 "主辦方登入"
3. 輸入管理員密碼

### 用戶註冊
1. 訪問網站
2. 點擊 "用戶登入"
3. 輸入手機號碼或主辦方生成的代碼

## 🔧 開發設置

### 安裝依賴
```bash
npm install
```

### 運行開發服務器
```bash
npm run dev
```

### 構建生產版本
```bash
npm run build
```

### 本地預覽生產版本
```bash
npm run serve
```
